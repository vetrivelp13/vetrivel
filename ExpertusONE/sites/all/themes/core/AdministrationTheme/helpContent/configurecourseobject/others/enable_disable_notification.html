Notification:
The Notification menu contains the list of notifications that are available for a course and you can enable/disable them based on the business requirement for your course.
 

Notification Language

Select the desired language from the drop down to set the language in which the notifications need to be triggered for this course.

Notify on Enrollment

Select to allow learners to get notifications upon enrolling to any of the classes of this course.

Notify on Enrollment Cancellation

Select to allow learners to get notifications upon cancelling their enrollment from any of the classes of this course.

Notify on Class Cancellation

Select to allow the registered learners to get notified upon cancellation of any of the ‘Classroom’ or ‘Classroom (Virtual)’ classes of this course.

Notify on Waitlist Confirmation

Select to allow the waitlisted learners to be notified when a confirmed seat becomes available. This notification will be triggered to all waitlisted learners in the class and the confirmed seat will be given on a first come first serve basis only.

Notify on Class Re-Schedule

Select to allow the registered learners to be notified whenever the class schedule changes. Any change to class date (or) time (or) location will trigger this notification to the registered learners.

Notify ‘x’ days before Class start date

Select to allow the registered learners to receive a reminder email ‘x’ days before the class start date. By default 2 days will be set at the system level and the LMS Administrator can override the same based on the business requirement for this course.
 
  • Clicking on each of the notifications listed above will open an HTML enabled text box where the LMS Administrator can add notes that can be appended in the standard email template, designed for each Notification.
  • This HTML-enabled text box supports multiple languages and you can draft the notes in your desired language.
  • These notes are generally the course specific instructions/information that needs to be notified to learners as part of the notifications they receive.
  • After typing necessary notes in the HTML enabled text box, please click Save button to ensure the notes is associated to the required notification template.
