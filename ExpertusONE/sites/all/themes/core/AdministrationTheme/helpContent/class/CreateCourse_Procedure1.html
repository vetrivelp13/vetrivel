PROCEDURE: Validation (Ensuring that the required course already exists)
  1. Click on the Administration menu item from the Learning Central home page.
  2. Select Course/Class from the Catalog Administration section of the Administer web part.
  3. In the Search Course/Class section that appears, ensure that the Title radio button is selected.
  4. Enter the name of course that you wish to search for, in the text entry box underneath the radio buttons.

Search using the following search criteria to find the desired course:

  • Exact course name that you wish to search
  • Similar course name matching the requirements of the course to be created.
 
  1. Click the Search button.

Note: Alternatively, to search for a course, you can also use the Search web part located on the top-left hand side of the Administer main page.


OUTPUT:

  • The search results will be displayed.
  • If the search results display the desired course details, then proceed with Workflow # 2 (Create ILT class) of this scenario.
