PROCEDURE : Create ILT class object
  1. Click on the Administration menu item from the Learning Central home page.
  2. Select Course/Class from the Catalog Administration section of the Administer web part.
  3. In the Search Course/Class dialog box that appears, ensure that the Catalog Title radio button is selected.
  4. Enter the name of course based on which you wish to create the ILT class, in the text entry box underneath the radio buttons. Select the course if it appears in the auto-populate list below the text entry box, while you enter the course name.
  5. Click the Search button.

Note: Alternatively, to search for a course, you can also use the Search web part located on the top-left hand side of the Administer main page.
 
  1. Click the Edit button  besides the course, based on which you wish to create the ILT class.
  2. Click the Associated Classes link in the secondary configuration items category, if it is not selected by default.
  3. Click the Add Class button to move on to the next page and configure the class parameters.

Note: Alternatively, you can click on the Add Class link under the Related Links section in the top right corner on this page to create a new class.
 
  • The next page appears with an additional set of configuration parameters.
  • The Parent Details section on the top contains the Course Title and the Course Code.
  • The Is Blended option will be checked or un-checked based on the parameter settings done while creating the course. This field is not editable. To modify this parameter, refer to the workflow # 1 (Create Course) of this scenario.
 

Note:The mandatory fields are marked with an asterisk. However, the Class Title, Class Code, and Class Description fields will automatically get populated by the system, based on the course parameters.
 
  1. From the Language drop-down menu, select the desired language for the new ILT class.
  2. From the Delivery Type drop-down menu, select Classroom to create a new ILT class.
The Classroom delivery type adds:
  • Registration End Date field.
  • Cancellation Deadline Date field.
  • The tabs for adding Sessions to the newly created ILT class, and for generating Reports related to this class.
  1. In the Attachment URL field, enter the URL of a supporting document, if required for the class. A supporting document could be a location map, agenda etc. which learners may want to refer while registering to the class.
  2. Next, select a status for your new class from the Status drop-down menu.
 

The Status drop-down menu has three options to choose from:
  • Active: Selecting Active will allow learners to view & enroll to this class.
  • Inactive: Selecting InActive will hide the class from learners’ view and will not allow them to enroll to this class. LMS administrator can generate custom reports to filter out the Inactive classes using this status.
  • Draft: Selecting Draft will hide the class from learners view and will not allow them to enroll to this class. Classes marked as Draft will be in unpublished state and once the class owner gives an approval, LMS Administrator can publish it by marking it as Active
  • Cancelled: Selecting Cancelled will cancel the class and all active registrations to the class. Necessary notification will be triggered to registered learner, if the respective notification was enabled for the class..
  1. Click the calendar icon  besides Registration End Date field to enter the date from when learners can start registering for this class.
  2. Click the calendar icon  besides Cancellation Deadline Date field to enter the date by which learners can be allowed to drop their enrollment from this class. Beyond this date, learners will not be allowed to drop their enrollment.
  3. Use the Duration field to enter the duration of this class in Hours and Mins. fields.
  4. Click on the Plus sign  besides each menu item to open its configuration parameters
 
Notification:
The Notification menu contains the list of notifications that are available for a class and you can enable/disable them based on the business requirement for your ILT class. The settings for these notifications are inherited from the course; however, you can override them if required.
 

Notification Language

Select the desired language from the drop down to set the language in which the notifications need to be triggered for this class. By default English is the language selected but you can override it based on your requirements for this class

Notify on Enrollment

Select to allow learners to get notifications upon enrolling to any of the classes of this class.

Notify on Enrollment Cancellation

Select to allow learners to get notifications upon cancelling their enrollment from any of the classes of this class.

Notify on Class Cancellation

Select to allow the registered learners to get notified upon cancellation of any of the ‘Classroom’ or ‘Classroom (Virtual)’ classes of this class.
Notify on Waitlist Confirmation Select to allow the waitlisted learners to be notified when a confirmed seat becomes available. This notification will be triggered to all waitlisted learners in the class and the confirmed seat will be given on a first come first serve basis only.
Notify on Class Re-Schedule Select to allow the registered learners to be notified whenever the class schedule changes. Any change to class date (or) time (or) location will trigger this notification to the registered learners.
Notify ‘x’ days before Class start date Select to allow the registered learners to receive a reminder email ‘x’ days before the class start date. By default 2 days will be set at the system level and the LMS Administrator can override the same based on the business requirement for this class.
 
Pricing:
The Pricing menu allows you to define pricing parameters for a class and specify currency for the price set. The settings for these pricing parameters are inherited from the course; however, you can override them if required.
 

Class Fee

Enter the amount to be paid by learners in order to register for this class.

Cancellation Fee

Enter the amount to be paid by learners when they cancel their enrollment.

Late Cancellation Fee

Enter the amount to be paid by learners when they cancel their enrollment after Cancellation Deadline date.

No Show Fee

Enter the amount to be paid by registered learners who don’t turn up for the class on the scheduled dates.
Currency Type Select a currency type from this drop-down menu. Available currencies: Euros, Indian Rupees, US Dollars.
The set currency will apply to all the price fields stated above.
No. of Training Units Enter the equivalent number of training units that denotes the price of this course.
Note: This is an alternate way of defining price for a course that doesn’t involve setting amount & currency. Ensure to leave other fields under the Pricing menu blank while using this option.
