Suggested Reading:

 

The Suggested Reading tab allows you to modify information about additional resources, which are related to this course. This information acts as additional reference to learners who want to learn more, in addition to what they would learn in this course.

 
 
  • Click Add Suggested Reading to add resource information

Type

The Type drop-down menu allows selecting the resource as a book, video or a URL.

URL

 The URL field captures the resource location that a learner can access.
  • Click Done to add the suggested reading.
  • Click Cancel to cancel the suggested reading creation.

Note: Multiple suggested reading resources can be added in similar way.
  • Click Edit besides the relevant resource entry to modify the entered parameter details.
  • Click Delete besides the relevant resource entry to remove the suggested reading from the list.

Note: You can click Preview above the configuration tabs to preview of all the parameters set for this course. LMS Administrator can use this information as a checklist to verify if the course has been configured as per the requirements defined by the course owner. The preview option triggers a new pop-up window with details, which you can either print or export as a PDF document using the functionalities provided in the pop-up window itself.
 

OUTPUT:

The desired course in LMS is modified as per the requirements.
