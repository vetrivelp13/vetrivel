PROCEDURE: ADD SESSION DETAILS
  1. Ensure that the Session Details tab is selected in the configuration area below the Duration field located in the class configuration section.
  2. Click the Add Session button for adding sessions to your virtual class.

Note: Each session requires entering values for Capacity Min, Capacity Max, WaitList Seats, Type, Start Date & Time, End Date & Time, URL, Time Zone, and Instructor. These parameters are explained below.
 

Capacity Min

The minimum number of attendees required to conduct this virtual class

Capacity Max

The maximum number of attendees that can be accommodated in this virtual class

WaitList Seats

The maximum number of attendees that can be kept in waiting list to join the virtual class, in case a registered attendee drops from the class

Type

This refers to the virtual class platform. Select appropriate value from the drop down list that contains Adobe, Centra, MSLiveMeeting, and WebEx

Start Date & Time

The start date and time of the virtual class

End Date & Time

The end date and time of the virtual class

URL

The URL that attendees will use to launch the virtual class

Time Zone

The time zone in which the class start and end time will be set. Using this information, attendees across multiple time zones can calculate the class timings applicable for their respective time zones. Based on the time zone preferences set in the learners’ profile in LMS, they will have option to see the virtual class timings in their local time zone (or) the actual time zone in which the class was setup.

Instructor

The name of the instructor who will deliver the virtual class
 
  1. Click Done button  to allow system to accept the filled-in parameters and add the session details to the virtual class.
  2. Click Delete button  to cancel the current add session procedure.
  3. Click Add Session button again to add more sessions to the virtual class.
  4. Click Save button when finished adding desired sessions to the virtual class, and allow the system to save the details set to this virtual class.

Note: The virtual class setup is complete and the class is now ready for learners to register.
