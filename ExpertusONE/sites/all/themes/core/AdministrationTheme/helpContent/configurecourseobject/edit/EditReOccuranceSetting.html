Re-Occurrence:

Re-Occurrence configuration menu enables LMS Administrators to modify the re-occurrence of the course by defining a period when it has to re-occur. This option is generally used for Compliance courses that the learners should complete periodically (for example, quarterly, half-yearly, or annually).

 
 

Re-Occurrence

If selected, enables re-occurrence of the course.

Valid From

The start date for course re-occurrence. Use calendar icon to select a date.

Valid To

The end date for course re-occurrence. Use calendar icon Â to select a date.

No. of Occurrence

Specifies the number of times the course re-occurs.

Auto Registration

Allows learners to automatically register for all subsequent occurrences of this course.
