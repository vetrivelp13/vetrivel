GENERATING CLASS ROSTER
  1. Click the Roster button  against the course or class for which you want to generate a roster report.
  2. The course or class roster pop-up window will display the Full Name, User Name, Email ID, Class Name, Registration
    Date, and Registration Status of the learners.

The registration status of the learners can be of the following types:

  • In Progress: This status represents successful registration of a learner and has a confirmed seat.
  • Waitlisted: This status represents waitlisted registrations.
  • Cancelled: This status represents the learners whose registration has been cancelled.
  • Completed: This status represents successful completion of the course or class by the learner.
 
  1. The Course or Class Roster pop-up window consists of four buttons: Print, Export to Excel, Convert to PDF, and Close.

Button

Description

Print

Click Print to take a printout of the roster page.

Export to Excel

Click Export to Excel to export the roster page in Excel format.

Convert to PDF

Click Convert to PDF to save the roster page in PDF format.

Close

Click Close to close the roster pop-up window.
 

As you are aware, a course can have multiple classes and the registrations happen at class level. An alternative method to view course or class roster is as follows:
    1. Click the Edit button  against the required course.
    2. To view Course Roster:
      • Navigate to the Reports tab in the course details page.
      • From the reports list, select Course Roster to open the Course roster report.
      • The Course Roster pop-up window with the registration details pertaining to the Course is displayed.
    3. To view Class Roster:
      • Click the Associated Classes tab in the course details page.
      • From the list of classes displayed under Associated Classes tab, click the Edit button  against the required class.
      • Navigate to the Reports tab in the required class details page.
      • To open the roster report of the selected class, from the list of reports, click Class Roster link.
      • The Class Roster pop-up window with the registration details pertaining to the desired class is displayed.
 

OUTPUT:

Course/Class level Roster Report will be generated.
