Suggested Reading:
The Suggested Reading tab allows including information about additional resources which are related to this course. This information acts as additional reference to learners who want to learn more, in addition to what they would learn in this course.
 
  • Click the Add Suggested Reading button to add resource information.

Type

The Type drop-down menu allows selecting the resource as a Book, a Video or a URL.

URL

The URL field captures the resource location that a learner can access.
  • Click the Done button  to add the suggested reading.
  • Click the Cancel button  to cancel the suggested reading creation.
Note: Multiple suggested reading resources can be added in similar way.
  • Click the Edit button  besides the relevant resource entry to modify the entered parameter details.
  • Click the Delete button  besides the relevant resource entry to remove the suggested reading from the list.
Note: You can click the Preview button above the configuration tabs to get an overview of all parameters set for this course. LMS Administrator can use this information as a checklist to verify if the course has been configured as per the requirements defined by the course owner. The preview option triggers a new pop-up window with details, which you can print to a local printer, or export as a PDF document using the functionalities provided in the pop-up window itself.

OUTPUT:

  • After completing this procedure, the course should be configured and ready for an ILT class to be setup.
  • Proceed with the Workflow # 2: Create ILT Class.
Note: The procedures for this workflow are same as the Create Course workflow of an ILT class.
