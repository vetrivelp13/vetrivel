Business Rules:

 

Business Rules configuration menu defines parameters that affect the learners’ registration and completion of a course. You can enable or disable the business rules listed under this menu based on the business requirement for the course.

 
 

Manager Approval Required to Register

If selected, then the learner requires an approval from his/her manager’s for registering to this course.

Recurring Registration Allowed

If selected, allows the learners to register for multiple classes of this course.

Manager can mark complete

If selected, allows managers to mark completion for this course on behalf of learners.

Learner can mark complete

If selected, allows learners to mark completion for this course by themselves.

Display for learner

Is selected by default. If this check box is not selected, then it will hide the classes of this course from learners view and prevent any new registration to the classes of this course.
