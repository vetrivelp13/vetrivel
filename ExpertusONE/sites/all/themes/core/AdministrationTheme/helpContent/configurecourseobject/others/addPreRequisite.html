Pre-Requisite:
The Pre-Requisite tab captures the criteria that define the course or courses which the learners must complete prior to accessing this particular course.
  • Select either the Course Title or Course Code radio button.
  • Enter either the course name or the course code that you wish to search for, in the text entry box underneath the radio buttons. You can select the relevant entry if it appears in the auto-populated list below the text entry box, while you enter the course title or course code.
  • Click the Search button.
  • Click the Add button adjacent to the required course for adding it as a pre-requisite.
Note: Multiple courses can be added as pre-requisites in similar way.
  • Click the Delete button  adjacent to a course entry to remove the desired pre-requisite course from the list.
  • In the Pre-Requisites column, click the Required For link to select the profile fields relevant to the selected pre-requisites.
