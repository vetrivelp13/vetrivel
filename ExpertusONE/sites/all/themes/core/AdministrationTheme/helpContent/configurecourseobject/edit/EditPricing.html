Pricing:

 

Pricing configuration menu allows you to modify the pricing parameters for a course and specify currency for the price set.

 
 

Course Fee

Type the fees for the learners to register for this course.

Cancellation Fee

Type the fees for cancelling a confirmed enrollment to a class..

Late Cancellation Fee

Type the fees for cancelling a confirmed enrollment to a class after the cancellation deadline date.

No Show Fee

Type the fees for learners who don’t turn up for the class on the scheduled dates.

Currency Type

 Select the currency from this drop-down menu. Available currencies: Euros, Indian Rupees, US Dollars.
The selected currency will apply to all the price fields.
No. of Training Units Enter the equivalent number of training units that denotes the price of this course.
Notify ‘x’ days before Class start date Allows the registered learners to receive a reminder email ‘x’ days before the start date of the class. By default, the reminder email will be sent two days before the start of the class. The LMS Administrator can specify a different value based on the business requirement for this course.
 

Note: Training Units is an alternate way of defining price for a course that does not involve setting fees and currency. Ensure that other fields under the Pricing menu are blank when using Training Units.
