PROCEDURE: EDIT COURSE CONFIGURATION SETTINGS
  1. Click Plus next to each configuration menu item to view and edit the relevant parameters.

Re-Occurrence:

Re-Occurrence configuration menu enables LMS Administrators to modify the re-occurrence of the course by defining a period when it has to re-occur. This option is generally used for Compliance courses that the learners should complete periodically (for example, quarterly, half-yearly, or annually).

 
 

Re-Occurrence

If selected, enables re-occurrence of the course.

Valid From

The start date for course re-occurrence. Use calendar icon to select a date.

Valid To

The end date for course re-occurrence. Use calendar icon  to select a date.

No. of Occurrence

Specifies the number of times the course re-occurs.

Auto Registration

Allows learners to automatically register for all subsequent occurrences of this course.
 
Business Rules:

Business Rules configuration menu defines parameters that affect the learners’ registration and completion of a course. You can enable or disable the business rules listed under this menu based on the business requirement for the course.

 
 

Manager Approval Required to Register

If selected, then the learner requires an approval from his/her manager’s for registering to this course.

Recurring Registration Allowed

If selected, allows the learners to register for multiple classes of this course.

Manager can mark complete

If selected, allows managers to mark completion for this course on behalf of learners.

Learner can mark complete

If selected, allows learners to mark completion for this course by themselves.

Display for learner

Is selected by default. If this check box is not selected, then it will hide the classes of this course from learners view and prevent any new registration to the classes of this course.
 
Notification:

Notification configuration menu contains the list of notifications that are available for a course and you can enable/disable these notification based on the business requirement for the course.

 
 

Notification Language

Sets the desired language from the drop down list in which the notifications are sent for this course.

Notify on Enrollment

Allows learners to receive notifications on enrolling to any of the classes of this course..

Notify on Enrollment Cancellation

Allows learners to receive notifications on cancelling their enrollment from any of the classes of this course.

Notify on Class Cancellation

Allows the enrolled learners to be notified if any of the classes of this course is cancelled.

Notify on Waitlist Confirmation

Allows the waitlisted learners to be notified when their enrolment for the course is confirmed. This notification sent to all waitlisted learners in the class. Confirmation will be provided on first come first serve basis.
Notify on Class Re-Schedule Allows the enrolled learners to be notified when there is a change in the class schedule of the course. Any change to class date, time, or location will send this notification.
Notify ‘x’ days before Class start date Allows the registered learners to receive a reminder email ‘x’ days before the start date of the class. By default, the reminder email will be sent two days before the start of the class. The LMS Administrator can specify a different value based on the business requirement for this course.
 

Course Specific Message:

  • Clicking each of the notifications listed above will open a HTML-enabled text box where the LMS Administrator can add course specific message that is appended to the standard email template for each of the notifications.
  • The HTML-enabled text box supports multiple languages.
  • These notes are generally course specific instructions/information that needs to be notified to learners as part of the notifications they receive.
  • After typing the necessary message in the HTML enabled text box, click Save to ensure that the message is appended to the required notification.

 
 
Pricing:

Pricing configuration menu allows you to modify the pricing parameters for a course and specify currency for the price set.

 
 

Course Fee

Type the fees for the learners to register for this course.

Cancellation Fee

Type the fees for cancelling a confirmed enrollment to a class..

Late Cancellation Fee

Type the fees for cancelling a confirmed enrollment to a class after the cancellation deadline date.

No Show Fee

Type the fees for learners who don’t turn up for the class on the scheduled dates.

Currency Type

 Select the currency from this drop-down menu. Available currencies: Euros, Indian Rupees, US Dollars.
The selected currency will apply to all the price fields.
No. of Training Units Enter the equivalent number of training units that denotes the price of this course.
Notify ‘x’ days before Class start date Allows the registered learners to receive a reminder email ‘x’ days before the start date of the class. By default, the reminder email will be sent two days before the start of the class. The LMS Administrator can specify a different value based on the business requirement for this course.
 

Note: Training Units is an alternate way of defining price for a course that does not involve setting fees and currency. Ensure that other fields under the Pricing menu are blank when using Training Units.
 
Owner

Owner configuration menu lists the details of course owner, course manager, and learning coordinator of the course. You can type the first few characters of the names of the course owner, course manager, or the learning coordinator, and select the required name from the suggested results that appear below the text box.

 
 

Course Owner

Phone Number

The respective fax number of the course owner automatically gets filled-in upon selection of the course owner’s name.

The respective Email ID of the course owner automatically gets filled-in upon selection of the course owner’s name.

The respective phone number of the course manager automatically gets filled-in upon selection of the course manager’s name.

The respective fax number of the course manager automatically gets filled-in upon selection of the course manager’s name.

The respective Email ID of the course manager automatically gets filled-in upon selection of the course manager’s name.

The respective Email ID of the learning coordinator automatically gets filled-in upon selection of the learning coordinator’s name.
 
2. Select each tab below the configuration menus to edit data for parameters associated with each item.
 
Associated Classes:

The Associated Classes tab contains the list of all classes that were for this course. You can edit any of these classes by clicking Edit  against the desired class. Refer to Workflow 5B, 5C, or 5D to learn how to edit an ILT, Virtual, or an Online class.

 
 
Access Profile:

The Access Profile tab allows you to edit the audience who can view this course based on the learners’ profile parameters chosen here. Following are the learner profile parameters that can be defined as criteria for learners to view this course: Experience Level, Job Role, Geographical Hierarchy, and Industry.

 
 

Experience Level

Select the required experience levels of learners for accessing this course. LMS Administrators can allow learners of multiple experience levels to access this course.

Job Role

Select the required job roles of learners for accessing this course. LMS Administrators can allow learners with multiple job roles to access this course.

Geo Hierarchy

Select the geographical areas from where learners can access this course. See below for more details on how to select the required geographical areas.
  • Click Expand  to open the tree with available geographical locations of EMEA (Europe, Middle East and Africa), AMER (Americas), and APAC (Asia Pacific)
  • Each geographical location further has another expand button, allowing selection of countries in the respective regions.
  • Select individual regions/countries to be allowed for this course.
  • LMS Administrators can allow learners from multiple geographical locations to access this course.
  • Select the check box  besides the expand button to select all available options of a tree..
Select the industries from which learners can access this course. See below for more details on how to select the required industries
  • Click Expand  to open the tree with available industries.
  • Each industry further has another expand button, allowing selection of sub-industry of the respective industries.
  • Select individual industries to be allowed for this course.
  • LMS Administrators can allow learners from multiple industries to access this course.
  • Select the check box  besides the expand button to select all available options of a tree.
  • Click Add Row to create multiple access profile combinations.
  • Click Delete to delete the selected access profile combination.
  • Click Save to save the access profile configuration set.

Pre-Requisite:

The Pre-Requisite tab allows you to modify the criteria that define the courses, which learners must complete prior to accessing this course.

 
 
  • Select either the Course Title or Course Code radio button.
  • Enter either the course name or the course code that you wish to search for, in the text entry box underneath the radio
    buttons.You can select the relevant entry if it appears in the auto-populated list below the text entry box, while you enter
    the course title or course code.
  • Click Search.
  • Click Add adjacent to the required course for adding it as a pre-requisite.

Note: Multiple courses can be added as pre-requisites in similar way.
 
  • Click Delete adjacent to a course entry to remove the desired pre-requisite course from the list.
  • In the Pre-Requisites column, click the Required For link to select the profile fields relevant to the selected pre-requisites.

Equivalence:

The Equivalence tab allows you to modify the criteria that define the course or courses which are similar to this particular course. At the time of enrollment, learners will be notified about the equivalent course(s) associated to this course based on which they can decide whether to enroll or not. This also acts as a reference to the learners in case they wish to get more information in addition to the course for which equivalence is being set.

 
 
  • Select either the Course Title or Course Code radio button.
  • Enter either the course name or the course code that you wish to search for, in the text entry box underneath the radio
    buttons. You can select the relevant entry if it appears in the auto-populated list below the text entry box, while you enter
    the course title or course code.
  • Click Search.
  • Click Add adjacent to the required course for adding it as equivalence.

Note: Multiple courses can be added as pre-requisites in similar way.
 
  • Click Delete adjacent to a course entry to remove the desired equivalent course from the list.

Forum Topics:

The Forum Topics tab allows the LMS Administrator to modify the forum for learners to interact.

 
 
  • Click Add Forum Topic to create a new forum topic.
  • Enter the Subject of the new forum topic. A default subject, customized for your course automatically gets filled-in by the
    system. Edit if required.
  • Enter the Body of the new forum topic. A default body, customized for your course automatically gets filled-in by the system.
    Edit if required.
  • Click Done to create the new forum topic.
  • Click Cancel to cancel the new forum topic creation.

Note: Multiple forum topics can be added similarly.
 
  • Click Edit besides the relevant forum topic to modify the entered parameter details.
  • Click Delete besides the relevant forum topic to remove it from the list.
User Tags (Folksonomy):

The Folksonomy tab allows the LMS Administrator to edit user tags as key words, which can be used by learners to search for this course.

 
 
  • Enter a desired key word in the Enter Your Tag text entry box.
  • Click Submit.

Note: The entered tags appear as Attached Tags.
 
Suggested Reading:

The Suggested Reading tab allows you to modify information about additional resources, which are related to this course. This information acts as additional reference to learners who want to learn more, in addition to what they would learn in this course.

 
 
  • Click Add Suggested Reading to add resource information

Type

The Type drop-down menu allows selecting the resource as a book, video or a URL.

URL

 The URL field captures the resource location that a learner can access.
 
  • Click Done to add the suggested reading.
  • Click Cancel to cancel the suggested reading creation.

Note: Multiple suggested reading resources can be added in similar way.
 
  • Click Edit besides the relevant resource entry to modify the entered parameter details.
  • Click Delete besides the relevant resource entry to remove the suggested reading from the list.

Note: You can click Preview above the configuration tabs to preview of all the parameters set for this course. LMS Administrator can use this information as a checklist to verify if the course has been configured as per the requirements defined by the course owner. The preview option triggers a new pop-up window with details, which you can either print or export as a PDF document using the functionalities provided in the pop-up window itself.
 

OUTPUT:

The desired course in LMS is modified as per the requirements.
