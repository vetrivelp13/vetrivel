Business Rules:
The Business Rules menu defines parameters that affect the learner’s registration and completion of a course. You can enable or disable the business rules listed under this menu based on the business requirement for your course.
 
Manager Approval Required to Register Select if the learner requires his/her manager’s approval for registering to this course.
Recurring Registration Allowed Select if the learners can register to multiple classes of this course.
Manager can mark complete Select if you wish to allow managers to mark completion for this course on behalf of learners.
Learner can mark complete Select if you wish to allow learners to mark completion for this course by themselves.
Display for learner Selected by default. Disabling this option will hide the classes of this course from learners view and prevent any new registration to the classes of this course.
