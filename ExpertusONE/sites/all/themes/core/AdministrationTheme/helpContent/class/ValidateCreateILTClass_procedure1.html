PROCEDURE: Validation (Ensuring that the required ILT class does not exist)
  1. Click on the Administration menu item from the Learning Central home page.
  2. Next, select Course/Class from the Catalog Administration section of the Administer web part.
  3. In the Search Course/Class section that appears, ensure that the Title radio button is selected.
  4. Enter the name of ILT class that you wish to search for, in the text entry box underneath the radio buttons.

Search using the following search criteria to find the desired class:

  • Exact class name that you wish to search
  • Similar class name matching the requirements of the class to be created.
 
  1. Click the Search button.

Note: Alternatively, to search for a class, you can also use the Search web part located on the top-left hand side of the Administer main page.


OUTPUT:

  • The search results will be displayed.
  • If the search result displays the desired class details, then contact the class owner to get a confirmation if the displayed class meets the business requirements; if not get necessary details to update in LMS , and then the ILT class is ready for registration.
  • If the search displays the message “No Records Found”, then create the desired class by referring to the following sections of this user guide for creating an ILT class:
