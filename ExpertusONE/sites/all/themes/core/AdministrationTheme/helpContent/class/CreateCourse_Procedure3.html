PROCEDURE : CONFIGURE COURSE OBJECT
  • For configuring a newly created course object in continuation of procedure # 2 “Create course object”, please start from step # 7 below.
  • To search for a course from the LMS home page, and then proceed with its configuration, start from step # 1 below.
 
  1. Click on the Administration menu item from the Learning Central home page.
  2. Select Course/Class from the Catalog Administration section of the Administer web part.
  3. In the Search Catalog dialog box that appears, ensure that the Catalog Title radio button is selected.
  4. Enter the name of course that you wish to search for, in the text entry box underneath the radio buttons. Select the course if it
    appears in the auto-populated list, while you enter the course name

Search using the following search criteria to find the desired course:

  • Exact course name that you wish to search
  • Similar course name matching the requirements of the class to be created.
 
  1. Click the Search button.
Note: Alternatively, to search for a course, you can also use the Search web part located on the top-left hand side of the Administer main page.
 
  1. Click the Edit button  besides the course that you wish to configure.
Note: The course details get displayed on the next page, followed by the configuration menus Re-Occurrence, Business Rules, Notification, Pricing, and Owner. Additionally there are several tabs displayed below the menus which allows further configuration to the course parameters.
 
  1. Click on the Plus sign  besides each menu item to open its configuration parameters.
Re-Occurrence:
The Re-Occurrence menu enables LMS Administrator to create a re-occurrence of the course by defining a period when it has to re-occur. This option is generally used for certain Compliance courses which need to be taken by learners periodically (e.g., quarterly, half-yearly, annually etc…).
 
Re-Occurrence Select to enable re-occurrence.
Valid From Click the calendar icon  to enter the Valid From date.
Valid To Click the calendar icon  to enter the Valid To date.
No. of Occurrence Enter the number of times course needs to re-occur.
Auto Registration Select to allow learners to auto-register for all subsequent occurrences of this course.
 
Business Rules:
The Business Rules menu defines parameters that affect the learner’s registration and completion of a course. You can enable or disable the business rules listed under this menu based on the business requirement for your course.
 
Manager Approval Required to Register Select if the learner requires his/her manager’s approval for registering to this course.
Recurring Registration Allowed Select if the learners can register to multiple classes of this course.
Manager can mark complete Select if you wish to allow managers to mark completion for this course on behalf of learners.
Learner can mark complete Select if you wish to allow learners to mark completion for this course by themselves.
Display for learner Selected by default. Disabling this option will hide the classes of this course from learners view and prevent any new registration to the classes of this course.
 
Notification:
The Notification menu contains the list of notifications that are available for a course and you can enable/disable them based on the business requirement for your course.
 

Notification Language

Select the desired language from the drop down to set the language in which the notifications need to be triggered for this course.

Notify on Enrollment

Select to allow learners to get notifications upon enrolling to any of the classes of this course.

Notify on Enrollment Cancellation

Select to allow learners to get notifications upon cancelling their enrollment from any of the classes of this course.

Notify on Class Cancellation

Select to allow the registered learners to get notified upon cancellation of any of the ‘Classroom’ or ‘Classroom (Virtual)’ classes of this course.

Notify on Waitlist Confirmation

Select to allow the waitlisted learners to be notified when a confirmed seat becomes available. This notification will be triggered to all waitlisted learners in the class and the confirmed seat will be given on a first come first serve basis only.

Notify on Class Re-Schedule

Select to allow the registered learners to be notified whenever the class schedule changes. Any change to class date (or) time (or) location will trigger this notification to the registered learners.

Notify ‘x’ days before Class start date

Select to allow the registered learners to receive a reminder email ‘x’ days before the class start date. By default 2 days will be set at the system level and the LMS Administrator can override the same based on the business requirement for this course.
 
  • Clicking on each of the notifications listed above will open an HTML enabled text box where the LMS Administrator can add notes that can be appended in the standard email template, designed for each Notification.
  • This HTML-enabled text box supports multiple languages and you can draft the notes in your desired language.
  • These notes are generally the course specific instructions/information that needs to be notified to learners as part of the notifications they receive.
  • After typing necessary notes in the HTML enabled text box, please click Save button to ensure the notes is associated to the required notification template.
 
Pricing:
The Pricing menu allows you to define pricing parameters for a course and specify currency for the price set.
 

Course Fee

Enter the amount to be paid by learners in order to register for this course.

Cancellation Fee

Enter the amount to be paid by learners when they cancel their enrollment.

Late Cancellation Fee

Enter the amount to be paid by learners when they cancel their enrollment after Cancellation Deadline date.

No Show Fee

Enter the amount to be paid by registered learners who don’t turn up for the class on the scheduled dates.

Currency Type

Select a currency type from this drop-down menu. Available currencies: Euros, Indian Rupees, US Dollars.
The set currency will apply to all the price fields stated above.

No. of Training Units

Enter the equivalent number of training units that denotes the price of this course.
Note: This is an alternate way of defining price for a course that doesn’t involve setting amount & currency. Ensure to leave other fields under the Pricing menu blank while using this option.

 
 
Owner:
The Owner menu captures the names and contact details of the various owners of a course, such as course owner, course manager and learning coordinator. To enter the names of the course owner, course manager and the learning coordinator, enter the first few characters of his/her name and select the required name from the suggested results that appear below the text input box.
 

Course Owner

Phone Number

Fax Number

Email

Enter the name of course owner. This person owns the course and the content.

The respective phone number of the course owner automatically gets filled-in upon selection of the course owner’s name.

The respective fax number of the course owner automatically gets filled-in upon selection of the course owner’s name.

The respective Email ID of the course owner automatically gets filled-in upon selection of the course owner’s name.

Course Manager

Phone Number

Fax Number

Email

Enter the name of course manager. This person is the program manager who owns the L&D initiative.

The respective phone number of the course manager automatically gets filled-in upon selection of the course manager’s name.

The respective fax number of the course manager automatically gets filled-in upon selection of the course manager’s name.

The respective Email ID of the course manager automatically gets filled-in upon selection of the course manager’s name.

Learning Coordinator

Phone Number

Fax Number

Email

Enter the name of learning coordinator. This person is responsible for logistics and other administrative tasks.

The respective phone number of the learning coordinator automatically gets filled-in upon selection of the learning coordinator’s name.

The respective fax number of the learning coordinator automatically gets filled-in upon selection of the learning coordinator’s name.

The respective Email ID of the learning coordinator automatically gets filled-in upon selection of the learning coordinator’s name.
 
  1. Select each configuration tab below the configuration menus to provide data for parameters associated with each item.
Note: Following tabs are seen below the menus and they allow further configuration to a course: Associated Classes, Access Profile, Pre-Requisite, Equivalence, Forum Topics, Folksonomy, Suggested Readings, and Reports. This Workflow does not cover Associated Classes and Reports. These are covered in the next workflow of this scenario.

Associated Classes:

The Owner menu captures the names and contact details of the various owners of a course, such as course owner, course manager and learning coordinator. To enter the names of the course owner, course manager and the learning coordinator, enter the first few characters of his/her name and select the required name from the suggested results that appear below the text input box.
 
The Associated Classes tab contains the list of all classes created under this course and it allows editing any of those listed
classes by clicking Edit  button against the desired class.
Access Profile:
The Access Profile tab allows you to define the audience who can view this course based on the learners’ profile parameters chosen here. Following are the learner profile parameters that can be defined as criteria for learners to view this course: Experience Level, Job Role, Geographical Hierarchy, and Industry..
 
 

Experience Level

Select the required experience levels of learners for accessing this course. LMS Administrators can allow learners of multiple experience levels to access this course.

Job Role

Select the required job roles of learners for accessing this course. LMS Administrators can allow learners with multiple job roles to access this course.

Geo Hierarchy

Select the geographical areas from where learners can access this course. See below for more details on how to select the required geographical areas.

  • Click on the Expand button  to open the tree with available geographical locations of EMEA (Europe, Middle East and Africa), Amer (Americas), and Apac (Asia Pacific).
  • Each geographical location further has another expand button, allowing selection of countries in the respective regions.
  • Select individual regions/countries to be allowed for this course.
  • LMS Administrators can allow learners from multiple geographical locations to access this course.
  • Select the check box  besides the expand button to select all available options of a tree.

Industry

Select the industries from which learners can access this course. See below for more details on how to select the required industries.

  • Click on the Expand button  to open the tree with available industries.
  • Each industry further has another expand button, allowing selection of sub-industry of the respective industries.
  • Select individual industries to be allowed for this course.
  • LMS Administrators can allow learners from multiple industries to access this course.
  • Select the check box  besides the expand button to select all available options of a tree.
 
    • Click Add Row button to create multiple access profile combinations.
    • Click Delete button to delete the selected access profile combination.
    • Click Save button to save the access profile configuration set.
 
Pre-Requisite:
The Pre-Requisite tab captures the criteria that define the course or courses which the learners must complete prior to accessing this particular course.
 
  • Select either the Course Title or Course Code radio button.
  • Enter either the course name or the course code that you wish to search for, in the text entry box underneath the radio buttons.
    You can select the relevant entry if it appears in the auto-populated list below the text entry box, while you enter
    the course title or course code.
  • Click the Search button.
  • Click the Add button adjacent to the required course for adding it as a pre-requisite.
Note: Multiple courses can be added as pre-requisites in similar way.
  • Click the Delete button  adjacent to a course entry to remove the desired pre-requisite course from the list.
  • In the Pre-Requisites column, click the Required For link to select the profile fields relevant to the selected pre-requisites.
 
Equivalence:
The Equivalence tab captures the criteria that define the course or courses which are similar to this particular course. At the time of enrollment, learners will be notified about the equivalent course(s) associated to this course based on which they can decide whether to (or) not enroll. This also acts as a reference to the learners in case they wish to get more information in addition to the course for which equivalence is being set.
 
  • Select either the Course Title or Course Code radio button.
  • Enter either the course name or the course code that you wish to search for, in the text entry box underneath the radio buttons.
    You can select the relevant entry if it appears in the auto-populated list below the text entry box, while you enter
    the course title or course code.
  • Click the Search button.
  • Click the Add button adjacent to the required course for adding it as equivalence.
Note: Multiple courses can be added as pre-requisites in similar way.
  • Click the Delete button  adjacent to a course entry to remove the desired equivalent course from the list.
 
Forum Topics:
The Forum Topics tab allows the LMS Administrator to create a forum for learners to interact.
The Forum Topics tab allows the LMS Administrator to create a forum for learners to interact.
  • Click the Add Forum Topic button to create a new forum topic.
  • Enter the Subject of the new forum topic. A default subject, customized for your course automatically gets filled-in
    by the system. Edit if required.
  • Enter the Body of the new forum topic. A default body, customized for your course automatically gets filled-in by the system.
    Edit if required.
  • Click the Done button  to create the new forum topic.
  • Click the Cancel button  to cancel the new forum topic creation.
Note: Multiple forum topics can be added in similar way.
  • Click the Edit button  besides the relevant forum topic to modify the entered parameter details.
  • Click the Delete button  besides the relevant forum topic to remove it from the list.
User Tags (Folksonomy):
The Folksonomy tab allows the LMS Administrator to create & associate user tags as key words which can be used by learners to search for this course.
 
  • Enter a desired key word in the Enter Your Tag text entry box.
  • Click the Submit button.
Note: The entered tags appear as Attached Tags.
 
Suggested Reading:
The Suggested Reading tab allows including information about additional resources which are related to this course. This information acts as additional reference to learners who want to learn more, in addition to what they would learn in this course.
 
  • Click the Add Suggested Reading button to add resource information.

Type

The Type drop-down menu allows selecting the resource as a Book, a Video or a URL.

URL

The URL field captures the resource location that a learner can access.
 
  • Click the Done button  to add the suggested reading.
  • Click the Cancel button  to cancel the suggested reading creation.
Note: Multiple suggested reading resources can be added in similar way.
  • Click the Edit button  besides the relevant resource entry to modify the entered parameter details.
  • Click the Delete button  besides the relevant resource entry to remove the suggested reading from the list.
Note: You can click the Preview button above the configuration tabs to get an overview of all parameters set for this course. LMS Administrator can use this information as a checklist to verify if the course has been configured as per the requirements defined by the course owner. The preview option triggers a new pop-up window with details, which you can print to a local printer, or export as a PDF document using the functionalities provided in the pop-up window itself.
 

OUTPUT:

  • After completing this procedure, the course should be configured and ready for an ILT class to be setup.
  • Proceed with the Workflow # 2: Create ILT Class.
Note: The procedures for this workflow are same as the Create Course workflow of an ILT class.
