PROCEDURE # 2: Create course object
  1. Click on the Administration menu item from the Learning Central home page.
  2. Select Course/Class from the Catalog Administration section of the Administer web part.
  3. Click on the Add Course button from the Search Course/Class dialog box that appears.

Note: The Catalog web part opens on the next page. Notice that the mandatory fields are marked with an asterisk.
 
  1. Enter the Course Title.
  2. Enter the Course Code as per the standard coding convention.
  3. Enter a Description for the new course that you wish to create.
  4. Next, select a status for your new course from the Status drop-down menu.

The Status drop-down menu has three options to choose from:
  • Active: Selecting Active will keep the course in published state and allow learners to register for classes associated with it. Courses with this option selected will appear in the respective search results for LMS Administrators only.
  • Inactive: Selecting InActive will discontinue the course and learners will not be able to register for any classes associated with it. Courses with this option selected will appear in the respective search results for LMS Administrators only. Also, they can generate custom reports to filter out Inactive courses using this status.
  • Draft: Selecting Draft will keep the course in unpublished state and does not allow learners to register for any classes associated with it. Courses with this option selected will appear in the respective search results for LMS Administrators only, and they will appear in custom reports as well. Once the course owner gives approval, LMS Administrator can publish the course by marking the status as Active.

 
 
  1. Click the Search button.

Note: Is Blended? option needs to be checked only while creating a Blended Course..
 
  1. Finally, click the Save button to create the Course.

OUTPUT:
  • The confirmation message will appear above the Course Title field, ensuring that the new course has been successfully created.
  • The menus Re-Occurrence, Business Rules, Notification, Pricing, and Owner will appear below the Status field, allowing the LMS Administrator to configure the course.
