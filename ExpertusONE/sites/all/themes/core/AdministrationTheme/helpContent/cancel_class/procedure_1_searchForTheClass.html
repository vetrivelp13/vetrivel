SEARCH FOR THE CLASS
  1. In the Administer web part, from the Catalog Administration section, select Course/Class.
  2. You can search for a class based on class title or class code. In the Search Course/Class web page, by default the Title radio button remains selected.
  3. Type the name of the class you want to cancel, in the text box underneath the radio buttons.

Note: If you want to search for a class based on code, select Code radio button, then type the class code, in the text box underneath the radio buttons.
 

Search using the following search criteria to find the desired class:

  • Exact class name that you wish to search
  • Similar class name matching the class cancellation requirements.
 
  1. Click the Search button.

Note: Alternatively, to search for a class, you can also use the Search web part located on the top-left hand side of the Administer main page.
