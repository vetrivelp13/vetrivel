VIEW CLASS ROSTER
  1. Click the Roster button Â against the class that you want to cancel.
  2. In the Class Roster pop-up window that is displayed, check if there are learners with the In Progress registration status.
    • If there are learners with In Progress registration status, then ensure that you enable the class cancellation notification while cancelling the class.
    • If there are no learners with In Progress registration status, then you can proceed with cancelling the class directly without verifying if the class cancellation notification is enabled.
  3. After viewing the registration details, click Close to close the Class Roster pop-up window.

Note: The Roster button will not appear if there are no registrations for the class. In such cases, proceed with procedure #3.
