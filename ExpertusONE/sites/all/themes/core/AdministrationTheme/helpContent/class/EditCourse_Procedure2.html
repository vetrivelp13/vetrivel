PROCEDURE: EDIT COURSE DETAILS
  1. Click Edit next to the course you wish to edit.

The details of the selected course including the configuration menus, such as Re-Occurrence, Business Rules, Notification, Pricing,
Owner, are displayed. In addition, several tabs are displayed below the configuration menus that allow further configuration to the
course parameters.
 
  1. Modify the Course Title, if required
  2. Modify the Course Code as per the standard coding conventions, if required.
  3. Modify the Course Description, if required.
  4. Select the new status of the course from the Status drop-down menu, if you need to change the course status.

The Status drop-down menu provides the following four options:
  • Active: Selecting Active will allow learners to view and register for the class associated with this course.
  • Inactive: Selecting Inactive will not allow learners to view or register for the classes that are associated with this class. The LMS Administrators can generate reports to filter out the inactive courses. This option is usually used to discontinue a course.
  • Draft: Selecting Draft will not allow learners to view or register for the classes that are associated with this course. The course will not be published when marked as Draft. Once the course owner gives an approval, the LMS administrator can publish the draft course by marking the status as Active.
  • Cancelled: This option is not applicable for courses.

 
  1. Ensure that the Is Blended? check box is not selected.

Note: The Is Blended? check box needs to be selected only for Blended Courses.
