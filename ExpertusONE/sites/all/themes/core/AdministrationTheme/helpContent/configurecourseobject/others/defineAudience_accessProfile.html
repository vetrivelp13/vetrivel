Access Profile:
The Access Profile tab allows you to define the audience who can view this course based on the learners’ profile parameters chosen here. Following are the learner profile parameters that can be defined as criteria for learners to view this course: Experience Level, Job Role, Geographical Hierarchy, and Industry..
 
 

Experience Level

Select the required experience levels of learners for accessing this course. LMS Administrators can allow learners of multiple experience levels to access this course.

Job Role

Select the required job roles of learners for accessing this course. LMS Administrators can allow learners with multiple job roles to access this course.

Geo Hierarchy

Select the geographical areas from where learners can access this course. See below for more details on how to select the required geographical areas.

  • Click on the Expand button  to open the tree with available geographical locations of EMEA (Europe, Middle East and Africa), Amer (Americas), and Apac (Asia Pacific).
  • Each geographical location further has another expand button, allowing selection of countries in the respective regions.
  • Select individual regions/countries to be allowed for this course.
  • LMS Administrators can allow learners from multiple geographical locations to access this course.
  • Select the check box  besides the expand button to select all available options of a tree.

Industry

Select the industries from which learners can access this course. See below for more details on how to select the required industries.

  • Click on the Expand button  to open the tree with available industries.
  • Each industry further has another expand button, allowing selection of sub-industry of the respective industries.
  • Select individual industries to be allowed for this course.
  • LMS Administrators can allow learners from multiple industries to access this course.
  • Select the check box  besides the expand button to select all available options of a tree.
    • Click Add Row button to create multiple access profile combinations.
    • Click Delete button to delete the selected access profile combination.
    • Click Save button to save the access profile configuration set.
