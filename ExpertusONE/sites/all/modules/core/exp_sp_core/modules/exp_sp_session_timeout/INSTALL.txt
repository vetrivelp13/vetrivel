INSTALL.txt file
================
$Id: INSTALL.txt,v 1.2 2009/04/17 14:05:51 jvandervort Exp $

1. Place the exp_sp_session_timeout folder in your drupal/modules folder (or if multisite and this 
  module is meant to be available to just one site in drupal/sites/www.YOURSITE.com/modules)

2. Enable the exp_sp_session_timeout module at administer > modules

3. Ensure the correct admin users have permissions to administer the exp_sp_session_timeout module at
  administer > people > permissions

4. Set-up the module at administer > configure > people > exp_sp_session_timeout
