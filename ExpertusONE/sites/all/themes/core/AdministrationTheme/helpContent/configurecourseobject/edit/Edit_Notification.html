Notification:

 

Notification configuration menu contains the list of notifications that are available for a course and you can enable/disable these notification based on the business requirement for the course.

 
 

Notification Language

Sets the desired language from the drop down list in which the notifications are sent for this course.

Notify on Enrollment

Allows learners to receive notifications on enrolling to any of the classes of this course..

Notify on Enrollment Cancellation

Allows learners to receive notifications on cancelling their enrollment from any of the classes of this course.

Notify on Class Cancellation

Allows the enrolled learners to be notified if any of the classes of this course is cancelled.

Notify on Waitlist Confirmation

Allows the waitlisted learners to be notified when their enrolment for the course is confirmed. This notification sent to all waitlisted learners in the class. Confirmation will be provided on first come first serve basis.
Notify on Class Re-Schedule Allows the enrolled learners to be notified when there is a change in the class schedule of the course. Any change to class date, time, or location will send this notification.
Notify ‘x’ days before Class start date Allows the registered learners to receive a reminder email ‘x’ days before the start date of the class. By default, the reminder email will be sent two days before the start of the class. The LMS Administrator can specify a different value based on the business requirement for this course.
 

Course Specific Message:

  • Clicking each of the notifications listed above will open a HTML-enabled text box where the LMS Administrator can add course specific message that is appended to the standard email template for each of the notifications.
  • The HTML-enabled text box supports multiple languages.
  • These notes are generally course specific instructions/information that needs to be notified to learners as part of the notifications they receive.
  • After typing the necessary message in the HTML enabled text box, click Save to ensure that the message is appended to the required notification.
