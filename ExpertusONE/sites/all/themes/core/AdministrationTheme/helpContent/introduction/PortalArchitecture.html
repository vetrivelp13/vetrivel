Introduction to SmartPortal™ LMS
The SmartPortal™ LMS is a robust and comprehensive learning management system (LMS) that empowers an organization to manage and deliver training to employees in various locations. The LMS provides complete solution in terms of delivering content, managing registrations, managing catalogs, managing records, and reporting. The LMS supports various course delivery types, such as classroom, virtual, online, and blended. The highlight of the LMS is that it has a 2-tier
The catalog structure in SmartPortal™ LMS is built on a two-tier architecture. Now, let’s see what each tier contains.
 
Product Architecture
 
The catalog structure in SmartPortal™ LMS is built on a two-tier architecture. Now, let’s see what each tier contains.
 
  2-Tier Catalog Structure – Standard Course
   
Tier 1 consists of course template creation, where you will capture high level information of the course, such as course title, course code, course description, pricing, and notification.
Tier 2 consists of class creation and defining the delivery type. The LMS supports classroom, virtual, and online delivery types. Note that, you can create a class only when a course template is created. The registrations will happen only at the class level. The information that you will capture for each delivery type will differ. For example, if the delivery type is ILT or virtual class, you will have to capture information such as date, time, location, and language. In case of online class, you will have to link the content at the class level. A course can have multiple classes of various delivery types. The learner can register for any class and complete the course to receive the credit.
 
Another characteristic of the SmartPortal™ LMS is that it supports Blended course.
 
  2-Tier Catalog Structure – Blended Course
   
What is a blended course?
As the name implies, a blended course is a combination of classroom, virtual, and online classes. As shown in the image, to create a blended course delivery, you will have to first create a course template and then a blended class. While creating a blended course you will need a sequence of classes that the leaner needs to take. The sequencing feature persuades the learner to take the classes only in the sequenced manner. For example, if the sequence is classroom, online, and then virtual class, the leaner will be able to take-up the course only in the prescribed sequence. The learner will not have the option of taking classes of his choice.
 
Using the SmartPortal™ LMS catalog feature you can create a language equivalent course by creating a course template first. The LMS also supports the delivery of online courses in various languages. This is similar to creating an online class.
