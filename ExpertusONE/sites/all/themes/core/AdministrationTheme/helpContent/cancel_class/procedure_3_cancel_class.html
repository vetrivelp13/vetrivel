CANCEL CLASS
  1. Click the Edit button Â against the class that you want to cancel.
  2. From the Status drop-down list, select Cancelled.
  3. To notify the registered learners about the class cancellation, follow the steps listed below:
    1. Select the Notification menu.
    2. Notification menu expands. Under the menu, ensure that the Notify on Class Cancellation check box is selected.
    3. If you wish to add any notes in the cancellation notification, click on Notify on Class Cancellation. A HTML-enabled text box is displayed where you can type the cancellation note. Click Save after inserting the notes text in the HTML enabled text box.
  4. Click Save to save the changes.
 

OUTPUT:

The desired ILT or Virtual class will be cancelled and a notification on the class cancellation will be sent to the registered learners (if applicable).
