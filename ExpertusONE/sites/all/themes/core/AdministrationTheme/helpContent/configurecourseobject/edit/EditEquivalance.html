Equivalence:

 

The Equivalence tab allows you to modify the criteria that define the course or courses which are similar to this particular course. At the time of enrollment, learners will be notified about the equivalent course(s) associated to this course based on which they can decide whether to enroll or not. This also acts as a reference to the learners in case they wish to get more information in addition to the course for which equivalence is being set.

 
 
  • Select either the Course Title or Course Code radio button.
  • Enter either the course name or the course code that you wish to search for, in the text entry box underneath the radio
    buttons. You can select the relevant entry if it appears in the auto-populated list below the text entry box, while you enter
    the course title or course code.
  • Click Search.
  • Click Add adjacent to the required course for adding it as equivalence.

Note: Multiple courses can be added as pre-requisites in similar way.
  • Click Delete adjacent to a course entry to remove the desired equivalent course from the list.
