PROCEDURE: SEARCH COURSE
  1. Click Course/Class in the Administer web part of the Administer home page.

The Search Course/Class web part is displayed.

  1. Ensure that the Title option is selected.
  2. Type the name of course that you wish to search for in the text box.

You can search using the following criteria to find the desired course:
  • Exact name of the course that you wish to search
  • Similar course names that match the requirements of the course to be edited

 
  1. Click Search.

Note: Alternatively, you can use the Search web part on the top-left of the Administer main page to search for a course.
 

The search results listing the courses that match the search criteria are displayed.
