PROCEDURE : Add session details
  1. Ensure that the Session Details tab is selected in the configuration area below the Duration field located in the class configuration section.
  2. Click the Add Session button for adding sessions to your ILT class.

Note: Each session requires entering values for Capacity Min, Capacity Max, WaitList Seats, Start Date & Time, End Date & Time, Location, Classroom, and Instructor. These parameters are explained below.
 

Capacity Min

The minimum number of attendees required to conduct this ILT class

Capacity Max

The maximum number of attendees that can be accommodated in this ILT class

WaitList Seats

The maximum number of attendees that can be kept in waiting list to join the ILT class, in case a registered attendee drops from the class

Start Date & Time

The start date and time of the ILT class
End Date & Time The end date and time of the ILT class
Location The geographical location (generally city) where the ILT class will be conducted
Room The actual room where the class will be conducted (e.g., conference room, training room, lab etc…)
Instructor The name of the instructor who will deliver the ILT class
 
  1. Click Done button  to allow system to accept the filled-in parameters and add the session details to the ILT class.
  2. Click Delete button  to cancel the current add session procedure.
  3. Click Add Session button again to add more sessions to the ILT class.
  4. Click Save button when finished adding desired sessions to the ILT class, and allow the system to save the details set to this ILT class.

Note: The ILT class setup is complete and the class is now ready for students to register.
